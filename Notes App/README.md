A complete note-taking application with full CRUD — create, read, update, delete. Notes persist in localStorage so they survive page reloads. Includes search, categories, and word count.

What are inside project!
Full note-taking app: create notes with title, content, and category.
Edit notes in place — click to edit, click away to save.
Delete with confirmation. Pin important notes to top.
Real-time search filters notes as you type.
Filter by category: All, Work, Personal, Ideas.
Word count and character count on each note.
All data persists in localStorage — reload the page, notes are still there.
